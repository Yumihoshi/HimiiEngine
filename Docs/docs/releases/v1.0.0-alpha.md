# HimiiEngine v1.0.0 Alpha

HimiiEngine 的首个公开 Alpha 版本：一套面向 **2D 游戏开发** 的集成引擎，包含 **C++ 运行时**、**可视化编辑器** 与 **C# 脚本系统**，可在同一工作流中完成场景编辑、资源管理、物理模拟与 Play 调试。

---

## 引擎架构

| 类别 | 说明 |
|------|------|
| **语言与构建** | C++17、CMake、vcpkg 依赖管理 |
| **核心框架** | Application / LayerStack 主循环、事件系统、日志（spdlog + 编辑器 Console） |
| **场景模型** | 基于 EnTT 的 ECS；实体—组件—系统；场景序列化为 YAML（`.himii`） |
| **渲染** | OpenGL（GLAD）、2D 批处理渲染、正交/透视相机、Framebuffer 离屏渲染、MSDF 字体、天空盒 |
| **物理** | Box2D 二维物理：刚体、盒/圆碰撞体、Tilemap 碰撞、脚本碰撞回调 |
| **脚本** | .NET 8 + CoreCLR 宿主；`ScriptCore` 提供 Entity、Input、Log、Physics2D 等 API；Inspector 字段序列化；异步 `dotnet build` |

---

## HimiiEditor 编辑器

集成开发环境，采用停靠式布局，主要能力包括：

| 模块 | 能力 |
|------|------|
| **场景编辑** | 层级树、Transform / 组件检视、视口 Gizmo、Edit / Play / Simulate 三种模式 |
| **资源管理** | Content Browser；纹理、场景、动画、Tilemap、C# 脚本等 |
| **2D 动画** | `.anim` 多命名逐帧动画；Animation Editor 图集选帧与时间轴 |
| **Tilemap** | TileMap Editor、TileSet、瓦片地图碰撞 |
| **脚本** | C# 编写游戏逻辑；Script Console 编译输出；Play 热运行与日志面板 |
| **项目** | `.hproj` 工程文件；`assets/` 资源目录与 AssetRegistry |

## 运行与分发

本 Release 包采用分层目录结构：

```text
HimiiEngine.exe          # 编辑器入口
editor_layout.ini        # 默认编辑器停靠布局（首次运行）
HimiiEngine/
  engine.hpck            # 引擎内置资源二进制包
  ScriptCore.dll         # C# 脚本宿主
  ScriptCore.runtimeconfig.json
```

- **HimiiEngine.exe**：编辑器主程序（Release 配置输出名）
- **HimiiEngine/**：引擎运行时数据目录
- **HimiiRuntime**（源码构建）：无 UI 运行时，可加载 `.hproj` 运行游戏

## 快速开始

1. 解压 Release 包至任意目录
2. 确认已安装 [.NET 8 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)
3. 运行 `HimiiEngine.exe`
4. 创建或打开 `.hproj` 项目
5. 在 Content Browser 中打开 `assets/scenes/*.himii` 场景，点击 **Play** 运行

更完整的入门说明见 [用户手册 · 快速开始](../UserManual/GettingStarted.md)。

## Alpha 说明

本版本为 **早期预览**，当前重点在 2D 游戏原型与内容生产。以下能力仍在路线图中或仅为基础实现：

- 3D 高级渲染与后处理栈
- 完整的项目一键打包管线（编辑器 **File → Build Project** 提供基础导出）
- 资源导入器与 Console 增强

API、资源格式与编辑器行为可能随版本迭代调整。欢迎通过 [Issue](https://github.com/HimiiFish/Himii-Engine/issues) 反馈问题与建议。
