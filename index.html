    HimiiEngine 文档

    欢迎来到 HimiiEngine 文档中心。

    HimiiEngine 是一套面向 2D 游戏开发 的集成引擎，包含 C++ 运行时、HimiiEditor 可视化编辑器与 C# 脚本系统。请使用左侧导航浏览文档。

    推荐阅读路径

    1. 快速开始 — 构建编辑器、创建项目、Play 运行场景
    2. 教程：2D 平台跳跃 — 从零完成可玩角色
    3. 编辑器与功能 — 动画、物理、脚本、打包等总览
    4. 脚本 API 参考 — C# API 查阅
